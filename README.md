commonform-annotations
---------------------

For this Common Form ...

```json
{
  "content":[
    {"reference":"Indemnity"}
  ]
}
```

... an annotation might be ...

```json
{
  "message": "The heading \"Indemnity\" is referenced, but not used.",
  "level": "info",
  "path": ["content", 0],
  "source": "commonform-lint",
  "url": null
}
```
